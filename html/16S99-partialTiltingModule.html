(partial) tilting module


Let A be an associative, finite-dimensional algebra over a field k. Throughout all modules are finite-dimensional.

A right A-module T is called a partial tilting module if the projective dimension of T is at most 1 (pd⁢T⩽1) and ExtA1⁢(T,T)=0.

Recall that if M is an A-module, then by add⁢M we denote the class of all A-modules which are direct sums of direct summands of M. Since Krull-Schmidt Theorem holds in the category of finite-dimensional A-modules, then this means, that if

M=E1⊕⋯⊕En

for some indecomposable modules Ei, then add⁢M consists of all modules which are isomorphic to

E1a1⊕⋯⊕Enan

for some nonnegative integers a1,…,an.

A partial tiliting module T is called a tilting module if there exists a short exact sequence

0→A→T′→T′′→0

such that both T′,T′′∈add⁢T. Here we treat the algebra A is a right module via multiplication.

Note that every projective module is partial tilting. Also a projective module P is tilting if and only if every indecomposable direct summand of A is a direct summand of P.
