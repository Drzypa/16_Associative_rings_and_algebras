an Artinian integral domain is a field


Let R be an integral domain, and assume that R is Artinian.

Let a∈R with a≠0. Then R⊇a⁢R⊇a2⁢R⊇⋯.

As R is Artinian, there is some n∈ℕ such that an⁢R=an+1⁢R. There exists r∈R such that an=an+1⁢r, that is, an⁢1=an⁢(a⁢r). But an≠0 (as R is an integral domain), so we have 1=a⁢r. Thus a is a unit.

Therefore, every Artinian integral domain is a field.
