faithful group action


Let A be a G-set, that is, a set acted upon by a group G with action ψ:G×A→A. Then for any g∈G, the map mg:A→A defined by

mg⁢(x)=ψ⁢(g,x)

is a permutation of A (in other words, a bijective function from A to itself) and so an element of SA. We can even get an homomorphism from G to SA by the rule g↦mg.

If for any pair g,h∈G g≠h we have mg≠mh, in other words, the homomorphism g→mg being injective, we say that the action is faithful.
