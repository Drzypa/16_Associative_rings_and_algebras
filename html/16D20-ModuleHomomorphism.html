module homomorphism


Let R be a ring and M,N left modules over R. A function f:M→N is said to be a left module homomorphism (over R) if

  1. 1.

    f is additive: f⁢(u+v)=f⁢(u)+f⁢(v), and

  2. 2.

    f preserves left scalar multiplication: f⁢(r⁢v)=r⁢f⁢(v), for any r∈R.

If M,N are right R-modules, then f:M→N is a right module homomorphism provided that f is additive and preserves right scalar multiplication: f⁢(v⁢r)=f⁢(v)⁢r for any r∈R.
