Prüfer domain


A commutative integral domain R is a Prüfer domain if every finitely generated nonzero ideal I of R is invertible.

Let RI denote the localization of R at R\I. Then the following statements are equivalent:

  • •

    i) R is a Prüfer domain.

  • •

    ii) For every prime ideal P in R, RP is a valuation domain.

  • •

    iii) For every maximal ideal M in R, RM is a valuation domain.

A Prüfer domain is a Dedekind domain if and only if it is Noetherian.

If R is a Prüfer domain with quotient field K, then any domain S such that R⊂S⊂K is Prüfer.
